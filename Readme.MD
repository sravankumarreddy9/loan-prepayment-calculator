# Loan Prepayment — Bank-Matching Calculator

A full-stack web app to simulate **loan prepayment** and visualize its impact on EMI, tenure, and total interest — designed to closely match **Indian bank amortization schedules** (like GIC Housing, HDFC, SBI, etc.).

---

# Live Demo

**Frontend:** [https://loan-prepayment-calculator-zeta.vercel.app](https://loan-prepayment-calculator-zeta.vercel.app)  
**Backend:** [https://loan-prepayment-calculator.onrender.com](https://loan-prepayment-calculator.onrender.com)

---

## Features

✅ Enter loan details (principal, EMI, interest rate, tenure, EMIs paid)  
✅ Add multiple prepayments (one or more months)  
✅ Choose to **Reduce Tenure** or **Reduce EMI**  
✅ See recalculated schedule, interest saved, and tenure reduction  
✅ Bar chart visualization (Principal vs Interest)  
✅ Export / Print friendly report  
✅ Deployed using **Vercel (frontend)** and **Render (backend)**  
✅ Built with **React.js**, **Express**, and **Chart.js**
